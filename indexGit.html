Mi primer trabajo con GIT

La característica de Git que realmente lo diferencia de casi todos los demás SCM es su modelo de ramificación. Git le permite y lo alienta a tener varias sucursales locales que pueden ser completamente independientes entre sí. La creación, fusión y eliminación de esas líneas de desarrollo lleva unos segundos.
